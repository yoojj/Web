# Web Server
= HTTP Server   
: 하드웨어 측면에서 웹 서비스와 관련된 정적 리소스를 저장하고 소프트웨어를 실행하는 컴퓨터   
: 소프트웨어 측면에서 클라이언트 요청을 받아 처리하고 결과를 응답하는 프로그램     
&nbsp; HTTP를 해석해 요청받은 서비스를 제공하고 문제가 있다면 오류 메시지를 제공하는 등 해당 요청에 응답하는 프로세스

- Google Web Server (GWS)
