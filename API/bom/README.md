# BOM  
Browser Object Model   
: 브라우저 객체 모델  
: 브라우저 요소를 제어하기 위한 인터페이스    
: 벤더사마다 구현하므로 표준이 존재하지 않음       


```bash
# host object   
Window (BOM)
 ├─ Navigator
 ├─ Screen
 ├─ History
 ├─ Location
 ├─ Frames
 └─ Document (DOM)
     └─ ...

# native object
JS Built-in Object
 └─ ...
```
