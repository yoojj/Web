# datalist
: 옵션 목록 기능을 지원하는 태그     


**포함 요소**   
- [option](./option.md)


```html
<label>
    <input type="text" list="datalist">
    <datalist id="datalist">
        <option value="1">1</option>
        <option value="2">2</option>
        <option value="3">3</option>
    </datalist>
</label>
```


**datalist vs select**   
datalist는 자동 완성 기능을 위한 옵션 목록을 제공하며     
select는 지정된 옵션 목록의 값만 선택 가능
